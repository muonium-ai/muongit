## Goal
Create a single VERSION file at repo root with value 0.1.0 and have Swift, Kotlin, and Rust builds read/embed this version consistently.

## Acceptance Criteria
- [ ] `VERSION` file exists at repo root containing `0.1.0`
- [ ] Rust `Cargo.toml` version matches VERSION; `MuonGitVersion::STRING` reads `0.1.0`

- [ ] Existing version tests pass with the new wiring
